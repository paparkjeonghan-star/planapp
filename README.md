학습 관리 플래닝 웹앱

목표
- 학생별 주간 학습 플랜을 설정하고, 주중 시간표 기반으로 과목별 공부 시간과 세부 계획을 자동 생성하는 완성 제품 수준의 웹앱을 개발합니다.

선택한 스택
- 프론트엔드: React + TypeScript + Vite, Tailwind CSS
- 백엔드: Node.js + TypeScript, NestJS (권장) 또는 Express + Prisma ORM
- DB: PostgreSQL (개발은 SQLite 가능)
- 인증/실시간: Supabase 권장 (Auth + Realtime)
- 호스팅: Vercel(프론트)/Render 또는 DigitalOcean App(백엔드)

다음 단계
1. 데이터 모델 및 API 설계 문서 검토
2. 백엔드 초기 골격(NestJS + Prisma) 생성
3. 프론트엔드 초기 골격(Vite + React) 생성

파일들: `docs/data-model.md`, `docs/api.md`
